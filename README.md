## Install

```
npm i ins-echarts -S
```

## Quick Start

```javascript
import Vue from 'vue'
import InsEcharts from 'ins-echarts'
import 'ins-echarts/lib/ins-echarts.css'

Vue.use(InsEcharts)

// or
import {
  Line,
  // ...
} from 'InsEcharts'
Vue.component(Line.name, Line)
```
